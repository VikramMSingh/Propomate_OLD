Creating Leads

To create a lead.
Go to CRM -> Leads -> Create Lead

To import leads from a file
Go to CRM -> Leads -> Import Leads
For importing a sample csv file is provided which can be used to create a lead.

Another option is to connect your website to our CRM.
This is in development

Assign Agent

To assign an agent

On the Leads page, just click on auto assign agent and it will assign the lead or leads to the active agent with the least workload.

Assign Category

The agent or the admin agent, can assign the lead to following 6 categories apart from the default category which is Unassigned New.
The six categories are:
1. Assigned
2. Qualified
3. Contacted
4. Needs Follow Up
5. Converted
6. Unconverted

Follow Up

You can add followups of your interaction with the lead. Either as text or file upload. This will be visible to both the assigned agent and the admin agent. To add a followup, just go the lead and click on add follow up. We do plan to integrate email, so you can directly add any conversation with the lead to the followup section. Admin agent can also provide feedback to the assigned agent.

Questions?

For any issues or new features. Feel free to reach out to us. We are currently working on bringing in more features and functionality to Propomate. Such as support request tracking where Customers/Leads can provide direct feedback to agent on the portal.
